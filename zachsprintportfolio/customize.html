Kind of cool, right? Well, it's a lot more work than it seems to be. In the background, a lot of things work together just for this little box:


  • The HTML file needs to read the script using the script tag. It also needs to hold the colorpickers and the submit button.
  • The CSS file has to actually make the box, telling the website just how big to make it and what the default look should be in case the colorpicker doesn't work.
  • The JS file does the heavy lifting when it comes to the DOM, registering the function, getting the element ID, and locally saving the colors that are picked when you click submit.
